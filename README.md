# STORM MULTI-ICO CONVERT
- Приложение автоматически создает из изображения иконки PNG размеров: 16x16, 32x32, 64x64, 128x128, 256x256 в 32 битах, затем создает из них один мульти-файл в ICO (PNG удаляются), пригодный для использования в приложениях.
- 3 темы оформления.
